# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-14

### Added
- Initial release of SD Card Monitor add-on
- Integration with sdmon v0.9.0
- Support for Apacer, Kingston, SanDisk, and Western Digital industrial SD cards
- Configurable scan interval (60-86400 seconds)
- Configurable device path
- Add delay option for Kingston cards requiring extra time
- JSON output to /share for Home Assistant integration
- Comprehensive documentation and configuration examples
- AppArmor security profile
- Multi-architecture support (armhf, armv7, aarch64, amd64, i386)
